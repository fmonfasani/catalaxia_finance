# Checklist — 56 BYMA-only para buscar en la CNV (aif2)

## Workflow por empresa
1. Buscar la empresa por nombre en la consulta publica de la CNV.
2. Abrir su presentacion de ESTADOS CONTABLES.
3. Copiar la URL (https://aif2.cnv.gov.ar/presentations/publicview/<UUID>).
4. Correr:  python procesar_cnv.py TICKER "URL" --cargar
5. Si identidad=0.00% -> cargado. Marcar [x] abajo.

(los [ya] estan cargados via IR; conseguir su URL CNV es opcional/mejor)
