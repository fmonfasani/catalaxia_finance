
# INSTRUCCIONES: DESCARGA MANUAL DE LISTAS COMPLETAS

## Para obtener lista COMPLETA de CEDEARs (200+):

### Opción 1: BYMADATA (Oficial)
URL: https://www.bymadata.com.ar/
Pasos:
1. Ir a la sección "CEDEARs"
2. Buscar opción para descargar/exportar lista completa
3. Si no existe, scroll y copiar todos los tickers
4. Guardar en CSV

### Opción 2: BYMA Data
URL: https://www.byma.com.ar/datos-publicos
1. Buscar "descarga de datos"
2. Descargar archivo Excel/CSV de CEDEARs
3. Procesar el archivo

### Opción 3: Contactar BYMA
Email: datos@byma.com.ar
Solicitar: "Lista completa de CEDEARs con tickers"

## Para obtener lista COMPLETA de acciones BYMA (50+):

### Opción 1: BYMA (Oficial)
URL: https://www.byma.com.ar/empresas-listadas
Pasos:
1. Filtrar por: Panel General + Panel Líder
2. Scroll y copiar todos los tickers
3. Guardar en CSV

### Opción 2: BYMA Datos Públicos
URL: https://www.byma.com.ar/datos-publicos
1. Descargar "Listado de Emisores"
2. Procesar el Excel/CSV

### Opción 3: Bloomberg/Refinitiv
Si tienes acceso, tienen lista actualizada de BYMA
